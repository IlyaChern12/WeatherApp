# WeatherApp

Проект представляет собой веб-приложение для просмотра прогноза погоды, выполненное в рамках лабораторной работы №4 по Web-программированию.

Приложение разработано с использованием чистого HTML, CSS и JavaScript без сторонних фреймворков. При первом запуске запрашивается геолокация пользователя и отображается прогноз погоды для текущего местоположения, при отказе доступен ручной ввод города. Реализовано добавление и удаление городов, обновление данных без перезагрузки страницы и сохранение состояния через localStorage. Интерфейс адаптивен и корректно отображается на мобильных и десктопных устройствах. Данные о погоде получаются через HTTP-запросы к публичному API. Проект задеплоен через GitHub Pages.

Сайт доступен по ссылке: https://ilyachern12.github.io/WeatherApp/
